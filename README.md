Лабиринт сокровищ

Представьте: вы — отважный исследователь в древнем лабиринте, 
полном хитрых коридоров, скрытых сокровищ и загадок, 
которые проверят вашу смекалку. 
Это не просто игра — это ваша собственная текстовая adventure-игра, 
где вы сами программируете правила.

Требования
Python 3.11+
Poetry 1.8.4+

Работа с проектом через poetry
poetry install — устанавливает зависимости из poetry.lock.
poetry run project — запускает игру.

Проект использует Makefile для упрощения команд.
make install — устанавливает зависимости из poetry.lock.
make project — запускает игру.
make lint — проверяет код на соответствие стандартам стиля с помощью Ruff.
make build — собирает дистрибутив проекта в папку dist/.
make package-install — устанавливает собранный пакет локально в систему.
make publish — выполняет тестовую публикацию пакета (dry-run).
